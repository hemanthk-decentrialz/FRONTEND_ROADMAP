# 🗺️ Frontend Development Roadmap

Welcome to my **Frontend Development Roadmap** repository! 👋

This repository documents my journey of learning modern frontend development through practice files, mini-projects, and complete applications. Each folder focuses on a different technology or project, helping me build a strong foundation step by step.

---

# 📂 Repository Structure

| Folder | Description | How to Run |
|--------|-------------|------------|
| **HTML_CSS** | Practice files covering HTML5, CSS3, Flex-box, Grid, Responsive Design, and animations. | Open `.html` files directly in any web browser or use **Live Server** in VS Code. |
| **JAVASCRIPT** | JavaScript fundamentals, ES6+, DOM manipulation, events, and mini-projects. | Open `.html` files in a browser or run `.js` files using Node.js (`node filename.js`). |
| **TypeScript-demo** | TypeScript practice files covering types, interfaces, generics, and classes. | Compile using `tsc filename.ts` then run the generated JavaScript file. |
| **[react-tutorial](./react-tutorial/README.md)** | ⚛️ React.js learning project with reusable components and hooks. | See the dedicated README for installation instructions. |
| **MediConnect** | Responsive frontend healthcare application demonstrating UI design principles. | See the project section below. |
| **[StudyFlow](./studyflow/README.md)** | 📚 Student Productivity Dashboard built with Next.js, React, TypeScript, and Tailwind CSS. | See the dedicated README for installation and usage. |
