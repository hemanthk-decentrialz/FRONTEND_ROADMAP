| Folder | Description | How to Run |

## 🚀 How to Run Projects

### General Frontend (HTML/CSS/JS)
1. **Locate** the project folder.
2. **Open** the main `.html` file in your web browser.
3. For a better development experience, use **Live Server** in VS Code.


### 🚀 MediConnect Project

This is a full frontend project. Here's how to run it based on its structure:

### If it's a Static Project (HTML/CSS/JS):
# Method 1: Direct Open
# Navigate to MediConnect folder and double-click index.html

# Method 2: Live Server (Recommended)
# Open the folder in VS Code and click "Go Live" from the status bar

### React Tutorial (`react-tutorial`)
The React app requires Node.js and npm. Please refer to its specific guide:
👉 **[React Tutorial README →](./react-tutorial/README.md)**

### TypeScript Demo (`typescript-demo`)
1. Ensure TypeScript is installed globally: `npm install -g typescript`
2. Compile a `.ts` file: `tsc filename.ts`
3. Run the output `.js` file: `node filename.js`
